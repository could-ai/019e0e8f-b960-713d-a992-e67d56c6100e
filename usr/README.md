# NovaForge AI

NovaForge AI is an advanced, premium AI coding platform designed to feel like a real startup product. Built with Flutter, it offers tools for generating Discord bots, entire websites, mobile apps, and other full-stack applications through natural language prompts.

## Features
- **AI Workspace**: Advanced coding assistant with multi-tab previews, real-time file exploration, terminal emulation, and chat interactions.
- **Discord Bot Builder**: Create fully functional Discord bots using quick templates (Economy, Moderation, Ticket systems) and export directly to ZIP or deploy.
- **Modern UI**: Clean, futuristic dark-mode aesthetic with neon accents and sleek typography, optimized across mobile, tablet, desktop, and web views.

## Tech Stack
- **Framework**: Flutter (Cross-Platform)
- **Language**: Dart
- **Design System**: Material 3 with Custom Themes, Google Fonts (Outfit), and Lucide Icons.

## Setup & Run
1. Ensure you have the Flutter SDK installed.
2. Clone this repository.
3. Run `flutter pub get` to install dependencies.
4. Use `flutter run` to launch the app on your preferred emulator or connected device.
